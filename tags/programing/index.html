Php, application/json 타입일시 POST data 가져오기
API 테스트 중 post data 가 안받아진다?! 평소 회사에서 사용하던 PHP 프레임워크는 Laravel 이었다. Laravel 에서 application/json의 content-type으로 request 를 받을시, Laravel 의 Reqeust 클래스를 이용하면 post data를 잘 만 받아 왔었다. 그런데 이번에 타팀 업무에 투입되어 Codeigniter 를 이용하여 Laravel 에서 했던 작업을 마이그레이션 해야 될 일이 있었다. 같은 PHP 기반 프레임워크 였기에 별다른 어려움 없이 API 를 개발하였고 연동하시는 분에게 확인을 부탁드렸다. 연동하시는 분께서 확인 후 오류가 있다고 말씀해 주셨다. 더보기
